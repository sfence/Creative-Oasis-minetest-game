# 石が多すぎる

良いテクスチャを作れる限りの石を追加するLuantiのmodです。TMSはFogblox、Hades Revisited、Mineclonia、Nodecore、VoxeLibreを含むほぼ全てのLuantiサブゲームをサポートしています。基本的に、ワールド生成があればこのmodがサポートします。

**バージョン1.4.0** - 現在**91種類の石**（そのうち7種類が光る）を特色とし、合計**2,074ノードと1アイテム**です。約半分が丸石バリエーションを持ち、すべて（アゲート、アンモライト、オパール、柱状バサルトを除く）がレンガと磨かれたブロックバリエーションを持ちます。階段、スラブ、壁はそれぞれのmodでオプション利用可能。ジオードはいずれかのジオードmodで利用可能。

各modには長所と短所があります。7つの光る石は、メガ洞窟がy-1,000以下で生成を開始すると見つけることができます。EmptyStarのおかげで、バイオーム全体でのはるかに良い石の分布を期待してください。一部の結晶は炉で焼くとひび割れ、各面で結晶を成長させ始めます。TMSは5.3まで完全に後方互換性がありますが、古いバージョンではログエラーが発生し、結晶は上向きのみになります。

すべての石は十分な掘削で見つけることができます。アンバー、アメジスト、すべてのカルサイト、セレスタイト、シトリン、エリスライト、ヘリオドール、ムーンストーン、プラシオライト、クォーツ、ローズクォーツ、スモーキークォーツ、すべてのトルマリン、ビビアナイトは、現実の対応物に合わせて透明です。
